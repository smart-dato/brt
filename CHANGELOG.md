# Changelog

All notable changes to `brt` will be documented in this file.

## 0.0.4 - 2026-03-09

### What's Changed

- Fix swapped mail/sms mappings in Alert value object
- Fix nullable openingHours causing TypeError in BrtSpecifics
- Fix parcelCount type from string to int in RequestInfos
- Add HasBody/HasJsonBody to UpdatePickupRequest so it can send data
- Fix `Italien` typo to `Italian` in Language enum
- Rename `Collection_Point`/`PickUp_Point` to PascalCase in Stakeholder Type enum
- Update README with real usage examples
- Remove unused Brt class, Facade, and BrtCommand stub

**Full Changelog**: https://github.com/smart-dato/brt/compare/0.0.3...0.0.4
